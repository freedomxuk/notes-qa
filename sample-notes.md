# 产品介绍

## 关于 Notes QA

Notes QA 是一个基于笔记的 AI 问答助手，帮助用户从大量笔记中快速找到所需信息。

### 核心功能
1. 笔记上传 - 支持 Markdown 和 TXT 格式
2. 智能问答 - 基于语义理解回答问题
3. 引用追溯 - 每个答案都可追溯到原文出处

### 技术栈
- Next.js - 前端框架
- Tailwind CSS - 样式
- Vercel - 部署平台
- OpenAI Embedding - 向量化
- MiniMax M3 - 大语言模型

# 研究笔记

## 2024年 AI 发展趋势

### 多模态理解
最新的 AI 模型已经能够理解图像、视频、音频等多种模态的信息，这使得 AI 应用场景更加广泛。

### 长上下文
随着上下文窗口的增大，AI 现在可以处理更长的文档，突破了传统 4K/8K 的限制。

M3 模型支持 1M token 的上下文窗口。
